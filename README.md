# E-commerce Financial Dashboard (Power BI)
An interactive Power BI dashboard designed to analyze financial performance by tracking sales, profit, and profit margins across products, categories, regions, and years. The dashboard provides actionable insights through dynamic visualizations and KPI monitoring, enabling data-driven business decisions.
 Project Overview
This project focuses on analyzing financial data using Power BI to monitor key performance indicators (KPIs) and uncover business insights. The dashboard allows users to explore sales, profit, and profit margin performance across different products, categories, regions, and time periods through interactive visualizations.
 Objectives
Track performance: Monitor sales and profit performance across different business segments.
Compare regions: Analyze and compare regional and category performance.
Identify trends:Spot financial trends and growth opportunities.
Support decisions: Provide data-driven insights for strategic decision-making.
 Tools Used
Power BI (DAX, Data Modeling, & Visualization)
Excel / CSV (Data Source)
Key Features
Interactive Dashboard:Equipped with slicers and filters for deep-dive analysis.
KPI Cards:Clear visibility for Sales, Profit, and Profit Margin.
Granular Analysis:Product and category performance breakdowns.
Geographical Insights:Regional performance comparison.
Temporal Trends: Yearly financial trend analysis.
